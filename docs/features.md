## Notifikasi UI Kustom

TherraBiz menggunakan sistem notifikasi UI kustom yang elegan dan tidak mengganggu, menggantikan `alert()` bawaan browser. Notifikasi ini dibangun dengan React Context API dan Framer Motion untuk animasi yang halus, memberikan umpan balik visual yang modern kepada pengguna.

## Mode Analitik Cerdas

*   **Basic Mode**: Menyediakan grafik standar dan ringkasan data bisnis.
*   **Advanced Mode**: Memanfaatkan AI untuk memberikan wawasan yang lebih mendalam dan analisis prediktif.
*   **Forecast Mode**: Berfokus pada proyeksi dan tren masa depan untuk membantu perencanaan strategis.

## Data Dummy

Fitur ini memungkinkan pengguna untuk mengaktifkan atau menonaktifkan data contoh (dummy data) untuk tujuan pengujian, demonstrasi, atau pengembangan. Ini sangat berguna untuk melihat fungsionalitas dashboard tanpa memerlukan data riil.
